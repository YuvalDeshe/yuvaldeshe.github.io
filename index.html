CURO Undergraduate Research - HeRo Lab | Spring 2026

The HeRo Lab has a heterogeneous robot swarm that can perform a few simple tasks. One of these tasks is an experiment called "Rendezvous," where the robots calculate the direction and speed they must move to reach a central point with all of the involved robots. Currently, the robots cannot perform any navigation around obstacles, nor can they replan their route if something in their environment changes.

My current plan is to:

  • Use the robots' camera server to track the robots' positions
  • Use an LLM to:
  • Plan initial robot routes
  • Navigate around obstacles
  • Replan routes when the environment changes
  • Replan routes when a human intervenes and changes the robot's course
  • Communicate the planned route to each robot using a behavior tree
